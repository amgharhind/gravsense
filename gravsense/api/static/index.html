📐 How to find calibration values

Reference width (cm)

The real-world width (in cm) that corresponds to the full width of your photo. You need one object of known size visible in the image to calculate this.

ref_width = known_object_cm ÷ fraction_of_image_width

Example: a truck bed is 230 cm wide and fills 80% of the image
→ reference width = 230 ÷ 0.80 = 288 cm

Truck camion-gravats-2tonnes.jpg: standard French benne ≈ 220–240 cm wide.
If it fills the full frame, use 230.

Pile height (cm)

The average vertical depth of the debris pile — used only to convert surface area → volume. Choose the method that fits your situation:

On-site measurement → tape measure at the pile edge
Visual estimate → 1 truck load of rubble ≈ 30–50 cm
Advanced → Depth Anything v2 (see README upgrade path)

Default of 30 cm is a reasonable first estimate for a single truck-load of mixed demolition debris.
